# Student Result Processing System

## Description
This is a console-based Python application that calculates:
- Total Marks
- Average Marks
- Pass/Fail Result
- Grade

## Concepts Used
- Functions
- Return Values
- Conditional Statements
- Loops
- User Input

## Features
- Accepts student details
- Takes 3 subject marks
- Calculates total and average
- Determines result and grade
- Allows multiple student entries
