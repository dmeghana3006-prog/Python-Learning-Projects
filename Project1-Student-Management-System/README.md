# Student Management System

## Description
This is a console-based Student Management System built using core Python concepts.
The application allows users to manage student records using a dictionary-based storage system.

This project was developed as a Phase 1 mini project.

---

## Technologies Used
- Python 3
- Core Python Concepts:
  - Dictionaries
  - Functions
  - Loops
  - Conditional Statements
  - User Input / Output

---

## Features
- Add student details (ID, Name, Marks)
- View all students
- Search student by ID
- Delete student by ID
- Menu-driven interface
- Handles invalid options

---

## Learning Outcome
- Working with dictionaries
- Creating modular functions
- Performing CRUD operations (Create, Read, Delete)
- Managing program flow using loops
